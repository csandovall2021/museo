# Museo - Sitio Web Interactivo

Sitio web interactivo desarrollado para la difusión y exhibición de contenido museográfico.

## Tecnologías Utilizadas

- **Backend:** PHP
- **Frontend:** HTML5, CSS3, JavaScript
- **Despliegue:** Servidor Institucional

## Características

- Interfaz interactiva y responsiva
- Navegación intuitiva
- Compatible con servidores institucionales

## Instalación

1. Clonar o copiar los archivos del proyecto en el directorio público del servidor institucional (ej. `public_html` o `www`).
2. Asegurarse de que el servidor tenga soporte para PHP habilitado.
3. Configurar los permisos de archivos y carpetas según sea necesario.
4. Acceder al sitio mediante la URL asignada por la institución.

## Uso

Una vez instalado, el sitio estará disponible para navegación pública a través del navegador web. No se requiere configuración adicional del lado del usuario.
